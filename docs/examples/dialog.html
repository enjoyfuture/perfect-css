Dialog 1

对话框

模板 A (标题+正文 大按钮)

Dialog 圆角为2px
按下时 Button 的背景色变为 P1
当 Button 文案字数大于等于4个字时，用该模板 A 样式

按钮的文案字色搭配方式
a 一般搭配搭配方式为 C6 + C2
b 当出现"删除" "退出" 等较警示的文案时，搭配方式为 C10 + C2

模板 B (标题+正文 多个按钮)

当有大于两个 Button 的情况时，不管文字字数都用该模板 B

模板 C (标题+正文 小按钮)

当 Button 文案字数小于4个字时
用模板 C

模板 D (图形+标题+正文)

当需要添加图形示意时，采用 D 模板样式
图形的高度建议高度不超过150px
下半部分规则同上

Android 特殊选择菜单栏

分带标题和不太标题两种 每次至少有两个选择条目
