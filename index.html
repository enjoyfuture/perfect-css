Perfect CSS

Perfect 是一款基于现代浏览器构建的 CSS 框架，运用最新 CSS 技术实现，给你带来完美的用户体验。 他也是一款未来的 CSS 框架。

Perfect 基于响应式布局、可定制主题实现，能够快速的搭建、开发应用。通过使用 CSS 变量能定制 各种主题，尽可能满足各种场景。 同时，Perfect 提供了大部分常用组件，并可以自定义组件样式。丰富的组件和定制化功能让你尽可能少些 CSS 样式， 只需拿来使用即可。
